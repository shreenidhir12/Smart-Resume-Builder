# Smart Resume Builder

A responsive web application that helps users create professional resumes quickly and easily.

## 🚀 Live Demo

https://shreenidhir12.github.io/Smart-Resume-Builder/

## 📌 Features

- Dynamic resume generation
- Professional resume preview
- Responsive design
- User-friendly interface
- Local storage support

- CSS
- JavaScript
